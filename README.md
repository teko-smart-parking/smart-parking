## docs

### linting

how to use:

1. install node LTS https://nodejs.org/en

2. install via command line `npm install -g markdownlint-cli`

#### lint docs

run this command `markdownlint docs/**/*.md -c .markdownlint.json`

## testing with Gauge

### install using npm
